# Roborto

A reproducible research playground for **embodied reinforcement learning**, studying one
question across two domains:

> What makes an embodied agent sample-efficient and generalizable - and does the answer
> transfer between **manipulation** and **navigation**?

- **Manipulation** - sparse-reward, goal-conditioned control on
  [Gymnasium-Robotics](https://robotics.farama.org/) (Fetch / Shadow Hand, MuJoCo).
- **Navigation** - visual navigation in [AI Habitat](https://aihabitat.org/)
  (PointNav / ImageNav).

This repository accompanies a series of technical reports. It is built for
reproducibility: every result is multi-seed with confidence intervals, every figure is
regenerated by a script, and every run records its exact config, git commit, and
package versions.

> **Status:** scaffold. See [`docs/PLAN.md`](docs/PLAN.md) for the full project plan and
> [`docs/lab_notebook.md`](docs/lab_notebook.md) for progress.

## Research spine

**Spine A - Representations & exploration for sample-efficient embodied control.**
The manipulation track studies goal relabeling (HER) and exploration; the navigation
track studies frozen visual foundation-model encoders versus trained-from-scratch
encoders; the synthesis asks whether "a good representation / good exploration" is
domain-specific or domain-general.

## Repository layout

```
configs/   Hydra configs - every hyperparameter lives here
roborto/   the installable package (envs, algos, models, buffers, ...)
scripts/   entry points (train, evaluate, sweep, make_figures)
tests/     unit tests for the non-trivial logic
docs/      the plan, setup, lab notebook, per-experiment design notes
reports/   LaTeX technical reports
```

## Quickstart

```bash
# manipulation environment (MuJoCo / Gymnasium-Robotics)
make setup-gym
conda activate roborto-gym

# run the unit tests
make test

# launch training on a chosen GPU (find an idle one with: nvidia-smi -L)
make train EXP=exp001_fetch_sac_her GPU=GPU-xxxxxxxx
```

Habitat lives in a separate conda environment - see [`docs/setup.md`](docs/setup.md).

## Reproducibility

Every figure in every report maps to one command in
[`docs/reproduce.md`](docs/reproduce.md). Results are aggregated across seeds with
`rliable` (interquartile mean + bootstrap confidence intervals).

## Tooling note

Baselines reuse Stable-Baselines3 (manipulation) and Habitat-Lab's DD-PPO (navigation);
the in-repo contributions are the HER relabeling study and the frozen-encoder
integration. Engineering scaffolding was assisted by Claude Code; the research design,
analysis, and writing are the author's own.
